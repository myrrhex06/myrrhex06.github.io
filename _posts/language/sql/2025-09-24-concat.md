## **CONCAT**
- 문자열을 결합할 때 사용함.

형식
```sql
SELECT
  CONCAT(string, string, ...)
FROM table_name;
```

예시
```sql
SELECT
  CONCAT(u.name, '-', u.email)
FROM users;
```

> 만약 `CONCAT`에 인자값으로 `NULL`이 들어가면 결과값 전체가 `NULL`로 떨어짐.
