## **ORDER BY**
- SELECT로 조회한 데이터를 정렬할 때 사용

형식
```sql
SELECT 
  column1, 
  column2,
  ...
FROM table_name
WHERE conditional
ORDER BY column1 [ASC | DESC];
```

- `ASC`: 오름차순
- `DESC`: 내림차순

예시
```sql
SELECT
  name,
  email
FROM user_info
ORDER BY user_seq DESC;
```

여러 컬럼에 정렬 설정도 가능

예시
```sql
SELECT
  name,
  email
FROM user_info
ORDER BY name DESC, email ASC;
```
- 이름을 내림차순으로 정렬한 후 이름이 같은 사용자가 있을 경우 이메일로 오름차순 정렬
