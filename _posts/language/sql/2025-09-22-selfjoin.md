## **셀프 조인(SELF JOIN)이란?**
- 하나의 테이블을 별칭을 서로 다르게 정의하여 조인하는 것을 말함.
  - 계층 구조 같은 관계를 표현할 때 주로 사용됨.

형식
```sql
SELECT 
  column_name,
  ...
FROM table_name a
JOIN table_name b
ON a.parent_id = b.id;
```

예시
```sql
SELECT
  u.name AS user_name,
  m.name AS manager_name
FROM users u
JOIN users m 
ON u.manager_id = m.user_id;
```
